# Opis Teleport Helper

Paste a copied `/opis` "Forced Chunks" row and get ready-to-use `/cofh tpx` teleport commands: the dimension, the exact anchor coordinates, and every claimed chunk.

**[Open the tool](https://willshaper.github.io/opis-teleport-helper/)**

## Usage

1. In-game, run `/opis` and open **Forced Chunks**.
2. `Ctrl+C` a row.
3. Paste it into the textarea.
4. Click **Copy** next to the command you need.

Everything runs client-side in your browser. No data leaves your machine.
